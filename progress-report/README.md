# Báo cáo tiến độ đồ án

## Mục đích
Thư mục này chứa các báo cáo tiến độ thực hiện đồ án theo từng giai đoạn.

## Hướng dẫn đặt tên file

Đặt tên file theo format: `YYYYMMDD_tuan-X_noi-dung.pdf`

Ví dụ:
- `20250101_tuan-1_khao-sat-yeu-cau.pdf`
- `20250108_tuan-2_thiet-ke-csdl.pdf`
- `20250115_tuan-3_phat-trien-giao-dien.pdf`

## Nội dung báo cáo nên bao gồm

1. **Công việc đã hoàn thành**
2. **Tiến độ thực hiện** (%)
3. **Khó khăn gặp phải** (nếu có)
4. **Kế hoạch tuần tiếp theo**
5. **Screenshots/Demo** (nếu có)

## Template báo cáo

```markdown
# Báo cáo tiến độ - Tuần X

**Ngày**: DD/MM/YYYY
**Sinh viên**: [Tên]
**MSSV**: [Mã số]

## 1. Công việc đã hoàn thành
- [ ] Công việc 1
- [ ] Công việc 2

## 2. Tiến độ
- Tổng quan: X%
- Chi tiết: ...

## 3. Khó khăn
- ...

## 4. Kế hoạch tuần sau
- ...
```

---

**Lưu ý**: Các file báo cáo cần được cập nhật đều đặn và commit lên repository.
